# 🦙 Ollama Llama Chatbot

A simple chatbot built using **Ollama** and **Llama models** for local AI inference.

## Features
- Chat with Llama models locally (no API cost)
- Powered by Ollama runtime
- Fast and private AI responses
- API integration

## Tech Stack
- NestJS
- Ollama
- Llama 3  model
- REST API
